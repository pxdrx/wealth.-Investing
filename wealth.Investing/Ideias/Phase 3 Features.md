# Phase 3 — High Impact Features

## Features Planejadas

### Dashboard Consolidado (Pro+)
- Visão multi-conta em uma tela
- Equity curve agregada
- PnL por período (dia, semana, mês)
- Comparação entre contas

### cTrader Parser
- Import de trades do cTrader
- Formato CSV
- Mesma lógica de dedup do MT5

### Onboarding Expandido (4 steps)
1. Display name (atual)
2. Seleção de plataforma (MT5/cTrader/manual)
3. Configuração de contas
4. Tour guiado

### Alertas Multi-canal
- Telegram bot
- Email digest
- Push notifications (PWA)

### Relatórios PDF
- Export de journal em PDF
- Gráficos incluídos
- Marca d'água wealth.Investing
