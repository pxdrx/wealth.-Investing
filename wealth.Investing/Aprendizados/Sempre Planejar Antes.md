# Sempre Planejar Antes

## Lição

> **NENHUM agente executa sem plano aprovado.**

## Processo

1. Receber tarefa
2. Escrever plano em `tasks/todo.md`
3. Apresentar plano para aprovação
4. Só então executar

## Por quê

- Evita retrabalho
- Garante alinhamento com o CTO
- Documenta decisões
- Facilita review
